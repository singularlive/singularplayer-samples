Singular Player SDK - Handling Sub-Compositions

This example shows how to hosts the Singular Player in an iFrame, load a composition and handle sub-compositions of a composition..

The composition is referenced by the unique id of the composition. Try following id's to load compositions with different layouts:

  • ID=8701: Name="Player-SDK-Diner"
  • ID=8702: Name="Player-SDK-Spring Forward (default)"
  • ID=8703: Name="Player-SDK-Blue"
  • ID=8704: Name="Player-SDK-Strawberry"
  • ID=8705: Name="Player-SDK-Lime"
  • ID=8706: Name="Player-SDK-Lighthouse"
