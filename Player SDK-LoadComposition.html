Singular Player SDK - Loading a Composition

last updated: 2017-06-23

This is an example for hosting the Singular Player in an iFrame and loading a HTML5 graphics composition.

Status:

The composition is referenced by the unique id of the composition. Try following id's to load compositions with different layouts:

  • ID=8856: Name="Player-SDK-Spring Forward (default)"
  • ID=8857: Name="Player-SDK-Diner"
  • ID=8859: Name="Player-SDK-Strawberry"
  • ID=8860: Name="Player-SDK-Lime"
  • ID=8861: Name="Player-SDK-Lighthouse"
  • ID=8862: Name="Player-SDK-Blue"
